# TypeScript + React

Simple user listing using react + typescript

### Getting Started

```
cd ./front-end
yarn i
```

and

```
cd ./back-end
yarn i
```

### Running the tests

```
cd ./front-end
yarn start
```

and

```
cd ./back-end
yarn dev:server
```


### API Back-end:
```
http://localhost:3001/users
```

### Fack-end:
```
http://localhost:3000
```
